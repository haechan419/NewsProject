# 브리핑 예약 데모 음성

`VITE_DEMO_VOICE=true` 시 음성 패널에서 번호키 1~4로 등록된 MP3를 음성 입력으로 전송한다.

## 사용 절차

1. 마이크 버튼 탭 → 번호키 1~4 중 입력 → 마이크 버튼 다시 탭
2. 선택한 키에 해당하는 MP3가 `analyzeVoice` API로 전송되어 STT·NLU 후 예약 처리

## 키–파일 매핑 (절대 시각)

| 키 | 파일명      | 추천 문장 (녹음) |
|----|-------------|------------------|
| 1  | brief_1.mp3 | 내일 아침 10시에 브리핑 보내줘 |
| 2  | brief_2.mp3 | 내일 오전 8시에 뉴스 메일 보내줘 |
| 3  | brief_3.mp3 | 오늘 15시 30분에 보내줘 |
| 4  | brief_4.mp3 | 오늘 HH시 MM분에 보내줘 (테스트 시마다 원하는 시각으로 녹음) |

- 4번: 테스트·캡처용. 예약할 시각(예: 몇 분 뒤)에 맞춰 「오늘 16시 5분에 보내줘」처럼 문장만 바꿔 녹음해 사용.
- 파일 미존재 시 해당 키 선택 시 로드 에러 발생.
