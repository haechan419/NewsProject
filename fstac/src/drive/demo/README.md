# 시연용 MP3 (VITE_DEMO_VOICE=true 시 사용)

키보드 **1~5**로 시연할 명령을 선택한 뒤, 마이크 버튼을 누르면 해당 MP3가 음성 명령으로 전송됩니다.

## 플레이리스트 방식 명령 (2025-02-01 업데이트)

| 키 | 파일명 | 용도 (예시 문장) |
|----|--------|------------------|
| 1 | `next.mp3` | 다음 ("다음으로 넘겨줘", "다음 뉴스") |
| 2 | `pause.mp3` | 일시정지 ("일시정지해줘") |
| 3 | `resume.mp3` | 재생 ("다시 재생해줘", "재생 계속해줘") |
| 4 | `help.mp3` | 도움말 ("도움말", "명령어 알려줘") |
| 5 | `stop.mp3` | 종료 ("종료해줘", "나가줘") |

## 제거된 명령
- **이어듣기** (`resume_choice.mp3`): 플레이리스트 방식에서는 이어듣기 기능이 제거됨

## 사용 방법
1. 드라이브 모드 진입
2. 플레이리스트 선택
3. 재생 중 키보드 **1~5** 중 하나 선택
4. 마이크 버튼 클릭
5. 선택한 명령의 MP3 파일이 음성 명령으로 전송됨

각 파일은 해당 문장을 녹음한 MP3로 준비하면 됩니다. 파일이 없으면 해당 키 선택 시 로드 에러가 납니다.
